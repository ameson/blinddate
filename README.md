# 漫画风格婚恋评价应用

一个基于 Vue 3 的漫画风格婚恋评价应用，帮助用户进行自我评估。

## 功能特点

- 多步骤信息收集
- 自动滚动表单导航
- 漫画风格的界面设计
- 动态评分系统
- 结果分享功能

## 技术栈

- Vue 3
- Vant UI
- Vue Router
- Vuex
- Composition API

## 开发环境设置

1. 克隆仓库
```bash
git clone [your-repository-url]
cd man
```

2. 安装依赖
```bash
npm install
```

3. 运行开发服务器
```bash
npm run dev
```

4. 构建生产版本
```bash
npm run build
```
